# Ping-0f-Death
HEAVILY multi-threaded dos/ddos ping tool

Installation :
```shell
git clone https://github.com/34zY/Ping-0f-Death.git
cd Ping-0f-Death/
python PoD.py
```

Usage : 
```shell
python PoD.py -i <Target IP> -t <Threads>
python PoD.py -l List_IP.txt -t <Threads>
```
